# homebrew-kryofetch — moved

kryofetch has moved into the main Krypton tap. Install it from there:

```bash
brew tap t3m3d/krypton
brew install kryofetch
```

Works on macOS (arm64) and Linux (x86_64). This standalone tap is retired;
the canonical formula now lives at
[t3m3d/homebrew-krypton](https://github.com/t3m3d/homebrew-krypton)
(`Formula/kryofetch.rb`).
